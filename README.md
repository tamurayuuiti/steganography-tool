# 画像ステガノグラフィー

React + Vite によるステガノグラフィーアプリケーションです。

LSB法を用いて、画像の見た目を変えずにファイルを埋め込み、隠されたファイルを取り出せます。ブラウザ上で完全に完結し、サーバへの送信は不要です。

---

## 主な特徴

- LSB（Least Significant Bit）法による画像ステガノグラフィー
- ファイルの埋め込み・抽出機能
- ドラッグ&ドロップによるファイル選択
- リアルタイムプログレス表示
- 埋め込み容量の自動検証
- ブラウザ内ですべての処理が完結し、画像やファイルはサーバへ送信されません

## 動作フロー

### ファイル埋め込み

1. 埋め込み対象の画像を選択
2. 隠したいファイルを選択
3. ファイルサイズ・画像容量の整合性を検証
4. LSB法によって画像のピクセルデータにファイルを埋め込み
5. 埋め込み済み画像をダウンロード

### ファイル抽出

1. 埋め込み済み画像を選択
2. 隠されたファイルのメタデータ（名前、サイズ）を抽出
3. 埋め込まれたデータを復号
4. ファイルとしてダウンロード

## ステガノグラフィー技術

### LSB (Least Significant Bit) 法

各ピクセルの RGBA チャンネルの最下位ビットにファイルデータを埋め込みます。

Alpha チャンネル（透明度）は画像の外観を大きく変えるため、RGB チャンネルのみを対象とします。

### データフォーマット仕様

```text
[NameLen (uint16)] [Name (UTF-8)] [DataLen (uint32)] [Data]
```

- NameLen: ファイル名の長さ（2バイト）
- Name: ファイル名（UTF-8エンコード）
- DataLen: ファイルデータの長さ（4バイト）
- Data: ファイルデータ本体

### 容量計算

埋め込み可能なデータサイズは、画像の解像度によって決定されます。

```text
埋め込み可能バイト数 ≈ (幅 × 高さ × 3) / 8
```

ただし、ヘッダー（ファイル名長 + ファイル名 + データ長）に必要な領域を差し引きます。
